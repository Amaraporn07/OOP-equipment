\# kanban-starter - Agent Rules

\## Stack

\- Runtime: Bun (not npm/node)

\- Server: Express.js

\- Port: 3000

\## Rules

\- NEVER use npm. Use bun only.

\- ALWAYS run `bun test` before claiming done.

\- Preserve original code logic unless asked.

\## Build Commands

\- Install: `bun install`

\- Run: `bun run dev`

\- Test: `bun test`
